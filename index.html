CSS Grid
Normal (Loser)

Instructions:

Use grid-columns & grid-rows to determine where the selected card is going.

Selectors: .deck, .blank, & .${suits}

Example:
.heart 2 {
grid-column 1 / 2;
grid-row 2 / 6;
}
